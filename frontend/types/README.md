# Generated API Types

This directory contains TypeScript type definitions auto-generated from the backend's OpenAPI 3.1.0 spec.

## Files

- `openapi.json` — The OpenAPI spec generated by `flask openapi generate --strict`
- `api.d.ts` — TypeScript types generated by `openapi-typescript` from the spec

## Regeneration

Run from the project root (requires Docker containers running):

    make generate-types

Regenerate after any backend change that modifies:
- Route handlers decorated with `@api_route`
- Pydantic request/response schemas in `backend/schemas/`
- Error code enums referenced by `@api_route`

## Do not edit manually

Both files are auto-generated. Manual edits will be overwritten on the next `make generate-types` run.
